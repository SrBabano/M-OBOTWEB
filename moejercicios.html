Balance de fuerzas

En esta sección se implementaron conceptos físico-matemáticos para conseguir que el robot tuviera un buen peso y distribución de componentes. Se buscó obtener la mayor eficiencia de peso en relación a las dimensiones del prototipo antes de ensamblar.

Balance de fuerzas

El prototipo inicial del chasis se diseñó con las dimensiones presentadas en la imagen para cumplir dos objetivos principales: Eficiencia, para no perder el rastro de la línea, y Velocidad, para que el robot haga el menor tiempo en el circuito.

Balance de fuerzas

Teóricamente, este fue el prototipo del chasis junto con cada uno de lo componentes. Se puede apreciar que buscamos no dejar muchos espacios sin utilizar, sin embargo este modelo resultó ser muy grande y tampoco tenía un buen acomodo de componentes per sirvió para concretar los modelos finales, puede revisarlos en la fase de Ensamblaje.
